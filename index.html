Azure Times

Welcome to Azure Times, the community blog for all things Azure!

About

Azure Times is a professional community in the global Cloud Practice map for everyone who is passionate about MS Azure Cloud. It intends to provide updates and insights from the real-world Azure evangelists, Cloud architects, and practitioners, establish experience and expertise sharing and engage people to active collaboration for having a world-class community of Azure Cloud professionals. This is the series of weekly online meetups, where we discuss a big scope of Cloud-related industry news, Microsoft product releases and updates, professional findings, and best practices that could be reusable within dedicated project teams, but not only. Meetings are going to be hosted in English.

Topics

We cover a wide range of topics, including but not limited to:

  • Azure Fundamentals
  • Azure DevOps
  • Azure Machine Learning
  • Azure Functions
  • Azure Logic Apps
  • Azure Kubernetes Service (AKS)

Contribute

We welcome contributions from the community. If you have an Azure-related topic you'd like to write about, please get in touch!
